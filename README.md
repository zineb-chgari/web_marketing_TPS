# Projet Social Media Analysis

Ce projet contient :
- un script Python d'envoi d'emails (`send_emails.py`)
- un notebook Jupyter d'analyse sociale (`social_media_analysis.ipynb`)

## Installation

1. Installer les dépendances :
pip install -r requirements.txt
